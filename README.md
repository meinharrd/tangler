# tangler

## Overview

Yet another iota tangle explorer.

## Requirements

This requires

* git
* go 1.7+


## Installation

    $ mkdir tmp
    $ cd tmp
    $ mkdir src
    $ mkdir bin
    $ mkdir pkg
    $ exoprt GOPATH=`pwd`
    $ go get github.com/utamaro/tangler
    $ go build
    $ ./tangler
